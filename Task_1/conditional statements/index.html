if Statement

Executes a block of code only if the condition is true.
int age = 20;
if (age >= 18) {
System.out.println("You are eligible to vote.");
}

if-else Statement

Executes one block if condition is true,otherwise executes another block. int number = 5;
if (number % 2 == 0) {
System.out.println("Even number");
} else {
System.out.println("Odd number");
}

if-else-if Statement

Checks multiple conditions one by one.
int marks = 75;
if (marks >= 90) {
System.out.println("Grade A");
} else if (marks >= 70) {
System.out.println("Grade B");
} else if (marks >= 50) {
System.out.println("Grade C");
} else {
System.out.println("Fail");
}

Nested-if

An if inside another if.
int age = 25;
boolean hasLicense = true;
if (age >= 18) {
if (hasLicense) {
System.out.println("You can drive.");
} else {
System.out.println("You need a license.");
}
} else {
System.out.println("You are too young to drive.");
}

Switch Statement

Used when checking one variable against multiple fixed values.
int day = 3;
switch (day) {
case 1:
System.out.println("Monday");
break;
case 2:
System.out.println("Tuesday");
break;
case 3:
System.out.println("Wednesday");
break;
default:
System.out.println("Invalid day");
}

Summary:

Control statements (if-else, loops, switch) are essential in programming to break the linear flow of execution, allowing programs to make decisions, repeat actions, and jump to specific code sections based on conditions. They provide flexibility, efficiency, and the ability to handle complex logic, interactive user input, and iterative tasks.
