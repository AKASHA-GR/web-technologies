2. Loops & Conditions

Java Core Foundations

Conditional Statements

Conditional statements are used to make decisions in a program.

Types of Conditional Statements

  1. if statement
  2. if-else
  3. if-else-if
  4. switch

Example Syntax


if(age >= 18){
   System.out.println("Adult");
}else{
   System.out.println("Minor");
}

Loops

Loops are used to repeat code multiple times.

Types of Loops

  1. for loop
  2. while loop
  3. do-while loop

Loop Example


for(int i=1; i<=5; i++){
   System.out.println(i);
}

Comparison Table

Type Category Use
if Condition Decision making
for Loop Repeating code
while Loop Condition-based repetition
