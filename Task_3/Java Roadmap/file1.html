What is a Variable?

A variable is a container used to store data in memory.

Variables allow programs to store and manipulate information.

Syntax


datatype variableName = value;

Types of Data Types

1. Primitive Data Types

These store simple values.

  1. int
  2. double
  3. char
  4. boolean
  5. float
  6. long
  7. byte
  8. short

2. Non-Primitive Data Types

These store objects or references.

  1. String
  2. Array
  3. Class
  4. Interface
